# Gym & Fitness Tracker 🏋️‍♂️

A responsive web application designed for fitness enthusiasts to log workouts and monitor health metrics, built as a self-directed project to master DOM manipulation and local data persistence.

## 🚀 Live Demo
https://pranaay2007.github.io/fitness-tracker/

## 🛠 Tech Stack
*   **Language:** Vanilla JavaScript (ES6+)
*   **Structure:** HTML5
*   **Styling:** CSS3 (Modern layout techniques)
*   **Data Persistence:** Browser LocalStorage API

## ✨ Key Features
*   **Workout Log:** Add, view, and delete daily workout entries.
*   **Progress Tracking:** Efficiently store and retrieve user data without a backend.
*   **Mobile-First Design:** Fully responsive layout for use inside the gym.
